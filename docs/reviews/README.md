# Review Notes

本目录存放 OutStakeV2 本地 review note 模板与草稿。

- review note 正文默认使用简体中文
- 结构以 `docs/reviews/TEMPLATE.md` 为准
- 后续的 review policy 和 gate 约束会在 `docs/process/` 中补齐
- `Commands run`、`Results`、`Verification evidence source` 仍归 `verifier` 负责；writer 侧字段轻量化不会减轻 `verifier` 对验证命令、验证 verdict 和相关证据的责任。
