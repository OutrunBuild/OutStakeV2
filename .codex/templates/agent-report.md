# Agent Report

> `required` 字段必须始终填写。
> `conditional` 字段仅在有相关内容且报告结论依赖它时必须填写；无相关内容时可省略或留空。
> 对需要多行内容的字段，请继续使用两个空格缩进。

- Role: required
- Summary: required
- Task Brief path: required
- Scope / ownership respected: required
- Files touched/reviewed: required
- Findings: conditional
- Required follow-up: conditional
- Commands run: conditional
- Evidence: conditional
- Residual risks: required
